FindASSIMP.cmake
  - There is no FindASSIMP cmake file installed on most Linux systems.

FindimageMagick.cmake
  - Although this file is included with many versions of CMake, older versions do not always find ImageMagick correctly.

FindGLEW.cmake
  - FindGLEW.cmake is installed on systems with cmake 2.8 and higher. However, at least one machine I regularly use only has 2.6 installed.

FindGLFW3.cmake
 - None of the systems I have tried provide a module to find GLFW3.

FindVRPN.cmake
  - There is no FindVRPN cmake file installed on most Linux systems.
